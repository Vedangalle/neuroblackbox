# NeuroBlackBox Demo Script

## 30-second opening

NeuroBlackBox is a local-first cognitive change memory for caregivers.

Families often notice changes before a doctor visit, but those observations are scattered and hard to explain clearly. NeuroBlackBox lets caregivers log observations, store them locally with Supermemory Local, and generate source-grounded summaries for clinician conversations.

It does not diagnose or treat. It organizes observations.

## Demo flow

### 1. Show local-first safety

Point to the banner:

NeuroBlackBox does not diagnose, treat, or predict Alzheimer’s, dementia, or any medical condition.

Then point to the Supermemory banner:

Supermemory SDK loaded. Local memory storage is available.

Explain:

Observations are stored locally through Supermemory Local.

### 2. Add a caregiver observation

Use this example:

Asked whether the family had dinner three times within 30 minutes, even after being reminded.

Set:

Type: repetition  
Severity: medium  
Source: caregiver

Click save.

Expected result:

Observation saved to CSV and Supermemory Local.

### 3. Ask about repeated questions

Ask:

Are repeated questions increasing?

Expected result:

- Supermemory Local returns relevant memory cards.
- Dashboard counters update.
- Doctor-prep summary reflects the observation.

### 4. Ask about speech pauses

Ask:

Are pauses increasing?

Expected result:

- Speech-related observations appear.
- Pause and word-finding mentions are visible in signal counters.

### 5. Show the high-impact moment

Ask:

What did we notice before the last bad episode?

Expected result:

NeuroBlackBox shows a before-episode analysis:

- latest high-severity episode
- observations before the episode
- repeated question signals
- speech pause signals
- routine disruption signals
- clinician-safe framing

### 6. Export summaries

Click:

Download doctor-prep summary

Then click:

Download before-episode analysis

## Closing line

NeuroBlackBox is not trying to be a doctor. It is trying to make sure the family walks into the doctor’s office with the right observations, in the right order, backed by source memory.
